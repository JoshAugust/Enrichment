## How to Read This

Job postings for GPU cluster engineering, ML infrastructure, and HPC roles are a strong buy-signal:
- Companies post these roles only when they're **operating or actively scaling** GPU infrastructure
- Titles like "GPU cluster engineer" or "HPC infrastructure engineer" imply **owned/leased clusters**, not just API usage
- "ML infrastructure" or "AI platform engineer" roles at non-AI-native companies signal **internal GPU fleet build-out**

4. **H100/H200/B200** mentioned by name → recent GPU purchases
5. Defense and pharma are now serious GPU buyers, not just hyperscalers
